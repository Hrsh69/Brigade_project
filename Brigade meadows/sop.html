Security & Standard Operating Procedures (SOP)

This page outlines all protocols to ensure safety, access control, emergency response, and community discipline at Brigade Meadows Phase-1.

Access Control & Visitor Management

All visitors must be registered using MyGate. Unauthorized persons should not be allowed entry. Delivery personnel must be verified and logged appropriately.

Vehicle & Parking Guidelines

Only authorized vehicles are allowed inside. Emergency access paths must always be clear. No parking in non-designated areas.

Gate Security & Patrol Duties

Security staff must maintain a presence at all entry/exit points and conduct regular patrols across common areas.

Emergency Protocols

In case of fire, medical emergency, or security breaches, guards must alert the EC, Estate Office, and if needed, local authorities.

CCTV Monitoring

Security should actively monitor surveillance cameras and report any suspicious activity to the management team.

Incident Reporting

Daily logs and reports must be submitted for every shift. Major incidents must be escalated and documented properly.

Security Staff Code of Conduct

  • Wear uniform and carry ID at all times.
  • No mobile phone usage during duty hours.
  • Polite, professional behavior is mandatory.
